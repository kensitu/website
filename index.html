Text Editor (C)


Implemented a simple text editor using doubly linked lists in order to efficiently distribute memory for large bodies of text.

Utilizes gap buffers to provide constant amortized time for text operations.

Lights Out Solver (C)


Uses the Breadth-First Search algorithm in conjunction with automatically-resizing, separate-chaining hash sets to efficiently find a solution to any board of the classic Lights Out game.

Simulated English Generator (Go)


Analyzes any number of input text files to create a frequency table comparing consecutive words to their succeeding word.

Subsequently generates English sentences using Markov Chains of any given order based off of the frequency table.
